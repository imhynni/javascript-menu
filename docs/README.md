# 기능 목록

## UI 로직

- 입력
  - [x] 코치의 이름을 입력 받는다.
  - [x] 각 코치마다 못 먹는 메뉴를 입력 받는다.
- 출력
  - [x] 서비스 시작 문구를 출력한다.
  - [x] 서비스 종료 문구를 출력한다.
  - [ ] 에러 상황 시 에러 문구를 출력한다.

## 도메인 로직

- [x] 요일마다 랜덤으로 카테고리를 고른다.
  - [x] 이미 한 주에 같은 카테고리가 2개 있을 경우 다시 고른다.
- [x] 각 코치가 요일별로 먹을 메뉴를 추천한다.
  - [x] 못 먹는 메뉴일 경우 다시 고른다.
  - [x] 이미 해당 음식이 있을 경우 다시 고른다.
